Bot Stats Panel

ABOUT THIS PROJECT:

This project is fully open sourced, it is designed to help developers monitor the resources being used by their bot. This panel has been made by Steel#7777.

SETUP:

1 - Edit the URL in "index.html" to the URL of your website, ignore the 2nd variable.

2 - In the file called ".htpasswd" add in a Bcrypt Username & Password.

3 - Upload the files to a host that supports PHP.
